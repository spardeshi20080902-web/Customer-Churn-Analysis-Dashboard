# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn data to identify patterns and factors that influence customer retention.

## Objective
To understand why customers leave a telecom service provider and create insights that can help improve customer retention strategies.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Data Analysis

## Analysis Performed
- Customer churn distribution analysis
- Churn analysis by contract type
- Churn trend based on customer tenure

## Key Insights
- Month-to-month contract customers show higher churn.
- Customers with longer contracts are more likely to stay.
- Early tenure customers require better retention strategies.

## Dataset
IBM Telco Customer Churn Dataset

## Project Outcome
Created an interactive dashboard to visualize churn patterns and generate business insights.
